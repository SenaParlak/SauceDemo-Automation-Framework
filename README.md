# SauceDemo Automation Framework

## About

This UI automation framework was developed using Java, Selenium WebDriver, Cucumber, TestNG, and Maven, following the Page Object Model (POM) design pattern to automate end-to-end UI testing for the SauceDemo e-commerce application.

## Technologies

* Java
* Selenium WebDriver
* TestNg
* Cucumber
* Maven
* Gherkin
* Page Object Model (POM)

## Tools
* Intelij IDEA
* Git
* GitHub

## Test Scenarios

* Valid Login
* Invalid Login
* Logout
* Add Product to Cart
* Remove Product from Cart
* Checkout Navigation
* Checkout Information
* Checkout Completion
* Back to Products
